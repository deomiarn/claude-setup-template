# Navbar Blocks (16)

Navigation bars and menu components for site headers. Provide intuitive site-wide navigation.

---

## navbar1

![screenshot](../images/navbar/navbar1.png)

Centered logo featuring structured content presentation and clear visual hierarchy. Displays menu items and logo for enhanced user engagement. Perfect for site header and conversion-focused applications. Responsive design with responsive structure and mobile-optimized layout. Best applied in scenarios requiring site header with professional visual impact.

**Install**: `pnpm dlx shadcn add @shadcnblocks/navbar1`
