# Pricing Blocks (35)

Pricing tables and tier comparisons for subscription services. Display plans, features, and cost structures.

---

## pricing1

![screenshot](../images/pricing/pricing1.png)

Three-column cards featuring structured content presentation and clear visual hierarchy. Displays highlighted tier and CTA buttons for enhanced user engagement. Perfect for subscription tiers and conversion-focused applications. Comparison-friendly design with responsive structure and mobile-optimized layout. Best applied in scenarios requiring subscription tiers with professional visual impact.

**Install**: `pnpm dlx shadcn add @shadcnblocks/pricing1`
