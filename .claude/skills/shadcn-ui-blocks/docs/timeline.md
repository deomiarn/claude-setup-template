# Timeline Blocks (14)

Timeline layouts for chronological information. Display events, milestones, and historical data.

---

## timeline1

![screenshot](../images/timeline/timeline1.png)

Centered axis featuring structured content presentation and clear visual hierarchy. Displays milestone dots and date markers for enhanced user engagement. Perfect for project roadmaps and conversion-focused applications. Organized design with responsive structure and mobile-optimized layout. Best applied in scenarios requiring project roadmaps with professional visual impact.

**Install**: `pnpm dlx shadcn add @shadcnblocks/timeline1`
