# Projects Blocks (24)

Project grid layouts for portfolio overviews. Display multiple projects in organized collections.

---

## projects1

![screenshot](../images/projects/projects1.png)

Masonry grid featuring structured content presentation and clear visual hierarchy. Displays featured images and filter buttons for enhanced user engagement. Perfect for work galleries and conversion-focused applications. Scannable design with responsive structure and mobile-optimized layout. Best applied in scenarios requiring work galleries with professional visual impact.

**Install**: `pnpm dlx shadcn add @shadcnblocks/projects1`
